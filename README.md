# Login/Signup Frontend (React)

This is the frontend repository for a login/signup system built with React.

## Table of Contents

- [About](#about)
- [Features](#features)
- [Installation](#installation)
- [Usage](#usage)
- [Contributing](#contributing)

## About

This repository contains the frontend code for a login/signup system. The system allows users to register for an account, log in, and access authenticated features.

## Features

- User registration
- User login
- Authentication
- Responsive design for various screen sizes

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/DevMehta22/01-Login-Signup-Frontend.git
   ```

2. Navigate to the project directory:

   ```bash
   cd 01-Login-Signup-Frontend
   cd frontend
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## Usage

Run the development server:

   ```bash
   npm start
   ```

Open your web browser and navigate to `http://localhost:3000` to view the frontend.
